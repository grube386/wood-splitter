# Requirements: Wood Splitter Premium Website

**Defined:** 2026-03-12
**Core Value:** Visitors instantly understand that Wood Splitter is the premium, safest, and fastest excavator-mounted log splitter — and can contact the company effortlessly.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Hero

- [ ] **HERO-01**: Video hero with video3.mp4 autoplay background (muted, looped, poster fallback)
- [ ] **HERO-02**: Product name "BOWS 20" and one-line value proposition visible above the fold
- [ ] **HERO-03**: Smooth scroll navigation from navbar to page sections

### Product

- [ ] **PROD-01**: Product showcase section featuring BOWS 20 with dual-side splitting, quick-attach wedge, safety, and power highlights
- [ ] **PROD-02**: Key advantages section with icon cards (safety, productivity, accessibility, durability)
- [ ] **PROD-03**: "Who is it for" section (foresters, farmers, home users, contractors)

### Brand

- [ ] **BRND-01**: About section with company story and 10th anniversary (2026) mention
- [ ] **BRND-02**: Dark industrial aesthetic using brand palette (red #D21C34 + dark #12161A)
- [ ] **BRND-03**: Responsive mobile-first layout

### Contact

- [ ] **CONT-01**: Contact form with name, email, and message fields with server-side email delivery
- [ ] **CONT-02**: Email address displayed in contact section and/or footer
- [ ] **CONT-03**: Social media links (Facebook, Instagram) in footer

### Internationalization

- [ ] **I18N-01**: All site content available in Slovenian, English, and German
- [ ] **I18N-02**: Language switcher UI component that changes language without page reload

### Performance

- [ ] **PERF-01**: Optimized video loading (poster image, preload metadata, no LCP blocking)
- [ ] **PERF-02**: Optimized images (WebP format, lazy loading for below-fold content)

## Out of Scope

| Feature | Reason |
|---------|--------|
| E-commerce / online purchasing | High-value B2B equipment requires quotes and dealer interaction |
| Blog / news section | Maintenance burden; stale content damages credibility |
| Trade fair schedule | Goes stale quickly; maintenance burden |
| IRP07 subsidy information | Owner decision — keep site focused on product/brand |
| Italian language (v1) | SL/EN/DE covers primary markets; add if analytics justify |
| Live chat widget | Requires staffing; unmanned chat worse than no chat |
| Product configurator | BOWS 20 is a single SKU; configurator adds false complexity |
| Newsletter signup popup | Industrial buyers are not newsletter subscribers; damages premium feel |
| Social media feed embeds | External JS weight, privacy concerns, unreliable APIs |
| Cookie consent modal | No analytics/tracking cookies in v1; minimal compliance burden |
